Frontend Overview 🎨

React 19 + Vite

🟢 Rapid and modern application engine.

🔵 Using latest Hooks, Fast Refresh, optimized build.

TypeScript

🟢 Secure code and fewer bugs.

🔵 Strict typing of props and API responses.

Zustand + TanStack Query

🟢 Fluid data management.

🔵 Server caching (Query) and lightweight global state (Zustand).

Tailwind v4 + Framer Motion

🟢 Beautiful design and animations.

🔵 JIT Compiler, complex declarative animations.

Backend Overview ⚙️

Go (Golang)

🟢 High-performance and robust language.

🔵 Native concurrency (Goroutines), strong static typing.

PostgreSQL

🟢 Reliable database.

🔵 Complex relations, ACID data integrity.

Deezer API

🟢 Why Deezer? More open than Spotify.

🔵 Public API accessible without complex OAuth for public data.

Security (JWT)

🟢 Secure login.

🔵 Stateless Tokens, Auth Middleware.
