نظرة على الواجهة الأمامية 🎨

React 19 + Vite

🟢 محرك تطبيق سريع وحديث.

🔵 استخدام أحدث الـ Hooks وبناء محسن.

TypeScript

🟢 كود آمن وأخطاء أقل.

🔵 تعريف صارم لبيانات الـ API.

Zustand + TanStack Query

🟢 إدارة بيانات سلسة.

🔵 تخزين مؤقت للبيانات وحالة عالمية خفيفة.

Tailwind v4 + Framer Motion

🟢 تصميم رائع ورسوم متحركة.

🔵 مجمع JIT ورسوم متحركة معقدة.

نظرة على الواجهة الخلفية ⚙️

Go (Golang)

🟢 لغة عالية الأداء وقوية.

🔵 ميزة التزامن الأصيلة (Goroutines).

PostgreSQL

🟢 قاعدة بيانات موثوقة.

🔵 تكامل البيانات وتوافق ACID.

Deezer API

🟢 لماذا Deezer؟ لأنها أكثر انفتاحاً من Spotify.

🔵 واجهة برمجية عامة سهلة الوصول.

الأمن (JWT)

🟢 تسجيل دخول آمن.

🔵 رموز Stateless و Middleware للهوية.
