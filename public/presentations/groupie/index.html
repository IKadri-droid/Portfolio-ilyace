Le Concept & La Vision

🎯 Pour Qui ?

Les fans inconditionnels, les "concert-goers", ceux qui vivent pour la musique live et ne veulent rater aucune date.

⚠️ Le Problème

L'information est dispersée. Sites officiels, billetteries, réseaux sociaux... C'est le chaos pour suivre une tournée complète.

💡 La Solution Groupie Tracker

Une interface unique, immersive et visuelle pour explorer les artistes, géolocaliser les concerts et tout savoir en un clic.

Zoom Frontend 🎨

React 19 + Vite

🟢 Le moteur de l'application, rapide et moderne.

🔵 Utilisation des derniers Hooks, Fast Refresh, Build optimisé.

TypeScript

🟢 Code sécurisé et moins de bugs.

🔵 Typage strict des props et des réponses API.

Zustand + TanStack Query

🟢 Gestion des données fluide.

🔵 Caching serveur (Query) et État global léger (Zustand).

Tailwind v4 + Framer Motion

🟢 Design magnifique et animations.

🔵 JIT Compiler, animations déclaratives complexes.

Zoom Backend ⚙️

Go (Golang)

🟢 Langage performant et robuste.

🔵 Concurrence native (Goroutines), Typage statique fort.

PostgreSQL

🟢 Base de données fiable.

🔵 Relations complexes, intégrité des données ACID.

Deezer API

🟢 Pourquoi Deezer ? Plus ouvert que Spotify.

🔵 API publique accessbile sans OAuth complexe pour les données publiques.

Sécurité (JWT)

🟢 Connexion sécurisée.

🔵 Tokens Stateless, Middleware d'authentification.
