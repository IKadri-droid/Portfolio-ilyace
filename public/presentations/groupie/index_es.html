Concepto y Visión

🎯 ¿Para quién?

Fans incondicionales, asiduos a conciertos, aquellos que viven por la música en vivo y no quieren perderse ninguna fecha.

⚠️ El Problema

La información está dispersa. Sitios oficiales, taquillas, redes sociales... Es un caos seguir una gira completa.

💡 La Solución Groupie Tracker

Una interfaz única, inmersiva y visual para explorar artistas, geolocalizar conciertos y saberlo todo en un clic.

Arquitectura Técnica Global

API Externa 🔗

  • Gestión de errores

Persistencia 💾

  • Relaciones Giras/Artistas

La Aplicación Central 💖

  • SPA React 19
  • Rendimiento (Vite)
  • Navegación fluida
  • Estado global (Zustand)

Seguridad y Autenticación 🛡️

  • JWT (JSON Web Tokens)
  • Protección de rutas
  • Hash de contraseñas (Argon2/Bcrypt)

Pagos 💵

  • Integración con Stripe
  • Webhooks seguros
  • Gestión de pedidos

Interfaz UI 🎨

  • Tailwind CSS v4
  • Framer Motion
  • Three.js / Globe

Infraestructura 🏗️

  • Docker (Contenedores)
  • Docker Compose (Orquestación)

Despliegue 🚀

  • CI/CD (Github Actions - Potencial)
  • Servidor Linux / VPS

Vista de Frontend 🎨

React 19 + Vite

🟢 Motor de aplicación rápido y moderno.

🔵 Usando los últimos Hooks, Fast Refresh, build optimizado.

TypeScript

🟢 Código seguro y menos errores.

🔵 Tipado estricto de props y respuestas de API.

Zustand + TanStack Query

🟢 Gestión fluida de datos.

🔵 Caché de servidor (Query) y estado global ligero (Zustand).

Tailwind v4 + Framer Motion

🟢 Diseño hermoso y animaciones.

🔵 Compilador JIT, animaciones declarativas complejas.

Vista de Backend ⚙️

Go (Golang)

🟢 Lenguaje de alto rendimiento y robusto.

🔵 Concurrencia nativa (Goroutines), tipado estático fuerte.

PostgreSQL

🟢 Base de datos confiable.

🔵 Relaciones complejas, integridad de datos ACID.

API de Deezer

🟢 ¿Por qué Deezer? Más abierto que Spotify.

🔵 API pública accesible sin OAuth complejo para datos públicos.

Seguridad (JWT)

🟢 Inicio de sesión seguro.

🔵 Tokens Stateless, Middleware de autenticación.
